Perguntas Frequentes

Encontre respostas para as dúvidas mais comuns sobre o sistema Front Access e os serviços digitais do Hospital das Clínicas.

Perguntas Gerais

O que é o Front Access?

O Front Access é um sistema desenvolvido pelo Hospital das Clínicas da Faculdade de Medicina da USP para facilitar o acesso digital aos serviços de saúde. Ele foi projetado para ser intuitivo e acessível a todos os usuários, independentemente de sua familiaridade com tecnologia.

Quem pode utilizar o sistema?

Todos os pacientes cadastrados no Hospital das Clínicas podem utilizar o sistema Front Access. Para acessar, é necessário ter um cadastro ativo e credenciais de acesso, que podem ser obtidas presencialmente em qualquer unidade do HC ou através do processo de cadastro online.

O sistema é gratuito?

Sim, o Front Access é totalmente gratuito para todos os pacientes do Hospital das Clínicas. Não há cobranças para criar uma conta, acessar informações ou utilizar qualquer funcionalidade do sistema.

Conta e Acesso

Como faço para criar uma conta?

Para criar uma conta no Front Access, você pode:

  1. Acessar a página "Iniciar" e clicar em "Criar Conta"
  2. Preencher o formulário com seus dados pessoais e documentos
  3. Verificar seu e-mail para confirmar o cadastro
  4. Definir uma senha segura para sua conta

Alternativamente, você pode solicitar a criação da conta presencialmente em qualquer unidade do Hospital das Clínicas.

Esqueci minha senha. Como recuperá-la?

Para recuperar sua senha:

  1. Acesse a página "Iniciar"
  2. Clique em "Esqueci minha senha"
  3. Informe seu CPF e e-mail cadastrado
  4. Siga as instruções enviadas para seu e-mail

Se não receber o e-mail de recuperação, verifique sua pasta de spam ou entre em contato com o suporte técnico.

Posso acessar minha conta em mais de um dispositivo?

Sim, você pode acessar sua conta do Front Access em múltiplos dispositivos, como computadores, tablets e smartphones. O sistema é responsivo e se adapta a diferentes tamanhos de tela para garantir uma boa experiência em qualquer dispositivo.

Agendamentos e Consultas

Como agendar uma consulta pelo sistema?

Para agendar uma consulta:

  1. Faça login em sua conta
  2. Acesse a seção "Agendamentos"
  3. Selecione a especialidade médica desejada
  4. Escolha a data e horário disponíveis
  5. Confirme seus dados e finalize o agendamento

Você receberá uma confirmação por e-mail e SMS com os detalhes da sua consulta.

Como cancelar ou remarcar uma consulta?

Para cancelar ou remarcar uma consulta:

  1. Acesse a seção "Meus Agendamentos"
  2. Localize a consulta que deseja alterar
  3. Clique em "Cancelar" ou "Remarcar"
  4. Siga as instruções na tela

Lembre-se: cancelamentos devem ser feitos com pelo menos 24 horas de antecedência para evitar penalidades.

Como funciona a teleconsulta?

As teleconsultas são realizadas através da plataforma segura do Hospital das Clínicas. Para participar:

  1. Acesse sua conta no horário agendado
  2. Vá para a seção "Minhas Consultas"
  3. Clique no botão "Entrar na Consulta"
  4. Permita o acesso à câmera e microfone quando solicitado

Recomendamos testar seu equipamento com antecedência e estar em um local silencioso e bem iluminado.

Funcionalidades do Sistema

Como aumentar o tamanho da fonte?

Para aumentar o tamanho da fonte, clique no botão "Aumentar Fonte" na seção de acessibilidade, localizada abaixo do conteúdo principal. Você pode clicar várias vezes para aumentar gradualmente o tamanho do texto até o limite máximo.

Como ativar o modo de alto contraste?

Para ativar o modo de alto contraste, clique no botão "Alto Contraste" na seção de acessibilidade. Isso alterará as cores do site para melhorar a legibilidade para pessoas com deficiências visuais. Clique novamente no mesmo botão para desativar o modo de alto contraste.

O site funciona em dispositivos móveis?

Sim, o Front Access é totalmente responsivo e funciona em qualquer dispositivo, incluindo smartphones e tablets. A interface se adapta automaticamente ao tamanho da tela para garantir uma experiência de usuário otimizada em qualquer dispositivo.

Como acessar meus resultados de exames?

Para acessar seus resultados de exames:

  1. Faça login em sua conta
  2. Acesse a seção "Meus Exames"
  3. Selecione o exame que deseja visualizar
  4. Clique em "Visualizar" ou "Baixar" para acessar o resultado

Os resultados ficam disponíveis no sistema assim que liberados pelo laboratório.

Segurança e Privacidade

Meus dados estão seguros no sistema?

Sim, o Hospital das Clínicas segue rigorosos protocolos de segurança e está em conformidade com a Lei Geral de Proteção de Dados (LGPD). Todos os dados são criptografados e armazenados em servidores seguros, com acesso restrito apenas a profissionais autorizados.

Quem tem acesso às minhas informações médicas?

Apenas você e os profissionais de saúde diretamente envolvidos no seu atendimento têm acesso às suas informações médicas. Cada acesso ao sistema é registrado e auditado para garantir a privacidade e segurança dos dados.
