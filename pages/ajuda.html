Navegação no Site

O site do Hospital das Clínicas foi projetado para ser intuitivo e fácil de usar. Aqui estão algumas dicas para ajudá-lo a navegar:

  • Utilize o menu de navegação no topo da página para acessar as diferentes seções do site.
  • A página inicial contém informações gerais e acesso rápido aos serviços mais utilizados.
  • Use os botões de navegação (Anterior/Próximo) para percorrer as páginas em sequência.
  • Caso tenha dúvidas específicas, consulte a seção de Perguntas Frequentes (FAQ).
  • Para entrar em contato conosco, utilize a página de Contato.

Recursos de Acessibilidade

Nosso site oferece diversos recursos para tornar a navegação acessível a todos os usuários:

  • Aumento e diminuição de fonte: Ajuste o tamanho do texto para melhor leitura.
  • Modo de alto contraste: Alterne para um modo de visualização com maior contraste entre texto e fundo.
  • Navegação por teclado: Todas as funcionalidades podem ser acessadas utilizando apenas o teclado.
  • Textos alternativos: Todas as imagens possuem descrições para leitores de tela.
  • Estrutura semântica: O site foi construído seguindo padrões de acessibilidade web.

Uso em Dispositivos Móveis

O site do Hospital das Clínicas é totalmente responsivo e pode ser acessado em qualquer dispositivo. Dicas para uso em dispositivos móveis:

  • No modo móvel, o menu de navegação se transforma em um menu hambúrguer (três linhas horizontais) no canto superior direito.
  • Toque no ícone do menu para expandir as opções de navegação.
  • Utilize os gestos de pinça para aumentar ou diminuir o zoom da página.
  • Gire o dispositivo para o modo paisagem para visualizar tabelas e conteúdos mais amplos.
  • Os formulários são otimizados para entrada de dados em telas sensíveis ao toque.
